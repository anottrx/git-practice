# Git 연습 및 공부하기

## 연습

### .gitignore

1. .env 파일을 업로드해서 push한다.
2. .env 파일을 제거하고, 앞으로도 해당 파일을 무시하기 위해 .env가 입력된 .gitignore을 push한다.
3. .env 파일을 새로 작성해도 2번 시점부터 .gitignore가 적용되어 .env는 올라가지 않는다. 다만 앞서서 올렸다가 지웠던 .env 기록은 계속 남는다.
